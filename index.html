Movie Database App
.NET Framework & C#

This is a course assignment where I developed a movie database using C#,GUI components and MS SQL Server.

How it works

You can add a movie to the database by filling out the form to the left of the interface and clicking on the ADD button. Input marked as "*" are required. The Movie Select COMBOBOX in the middle of the interface is used to select and view movie records from the database. Records selected are displayed(READ ONLY) on the form on the left side of the interface.
